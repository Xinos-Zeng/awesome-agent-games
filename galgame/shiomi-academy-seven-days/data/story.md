# 《汐見学園の七日間》— 世界观与剧情大纲

## 玩家可见开场简介

汐見学園の七日間
— 海边的学园，静止的灯塔，永无止境的七天 —

在悬崖之上，有一所名为「汐見学園」的私立高中。

昭和风格的旧校舍，远处灰白色的废弃灯塔，以及过于宁静的、仿佛被精心维护的空气。

据说，每到第七天的午夜零点，一切都会重置回第一天的清晨。

但你什么都不记得了。

你只知道，今天，是你转学的第一天。

你好，玩家。欢迎来到汐見学園的故事。

## 世界观

### 汐見学園（Shiomigahama Academy）

坐落于海边悬崖上的私立高中。校园建筑是昭和风格的旧校舍，远处能看到一座废弃的灯塔。
学生不多，每个年级只有两个班。氛围宁静，甚至可以说**过于宁静** — 像是被精心维护的标本。

### 核心设定：时间之环

二十年前，学园地下曾有一个名为「汀プロジェクト（Nagisa Project）」的研究设施，
由物理学家长谷川诚一（Hasegawa Seiichi）主持。实验目的是观测时间的微观折叠现象。

实验失控了。时空产生了裂隙，形成了一个**七天的时间循环**：
每到第七天的午夜零点，一切重置回第一天的清晨。

循环内的人**不会衰老、不会真正死亡**（死亡后也会在下个循环重置）。
但每次循环，**记忆会像沙漏里的沙一样流失** — 大多数人几个循环后就什么都不记得了，
以为自己一直在过"普通的一周"。

只有极少数人能保留跨循环的记忆碎片。这些人被称为「残响（Zankyou）」。

### 主角的真相

主角 **長谷川 蓮（Hasegawa Ren）** 是长谷川诚一的儿子。
二十年前实验失控时，他还是个孩子。父亲将他送进了时间循环的"安全区" — 也就是现在的汐見学園。
他已经在循环中度过了**远超自己认知的时间**，记忆几乎完全流失。
他以为自己是刚转学来的普通高二学生。

每个循环的第一天，他都会"转学"进汐見学園 — 这本身就是循环的一部分。

### 灯塔

校园远处废弃的灯塔是时间裂隙的物理锚点。
灯塔顶层有一个已经停止的钟 — 它停在 **00:00:00**，即循环重置的时刻。
灯塔地下连接着旧研究设施的残骸。

## 三位女主 — 与真相的联系

### 白瀬 雫（Shirase Shizuku）— "记忆的守望者"

**表面：** 安静寡言的图书委员。总是坐在图书馆最深处的座位上读书。
说话声音很轻，像是在怕吵醒什么人。眼神偶尔流露出与年龄不符的疲惫。

**真相：** 雫是循环中**记忆保留最完整的人**。她记得每一个循环 — 数百次，也许数千次。
她一直在图书馆里记录每次循环发生的事，用只有自己看得懂的符号编码。
她知道循环的真相，但每次试图告诉别人，那些人都会在下一个循环彻底遗忘。
她渐渐放弃了主动告知，转而默默守护。

**她的秘密：** 她的姐姐白瀬 響（Shirase Hibiki）是二十年前实验的助理研究员，
在实验失控时消失 — 不是死亡，而是被困在了时间裂隙的"另一侧"。
雫留在循环中，一半是为了寻找救姐姐的方法，一半是因为她害怕离开后自己也会遗忘姐姐。

**核心主题：** 记忆的诅咒与祝福。记住一切是痛苦还是幸福？
反复经历同样的七天，看着同样的人一次次忘记一切 — 这是一种什么样的孤独？

**线路核心矛盾：**
雫找到了打破循环的方法，但代价是**所有人都会失去循环中的全部记忆**。
包括她和主角之间在这无数次循环中积累的一切。
如果打破循环，她将回到一个没有人记得她的世界 — 包括主角。
如果保持循环，他们可以永远在一起，但这是一个虚假的永远。

---

### 御崎 琴音（Misaki Kotone）— "秩序的守护者"

**表面：** 学生会长。完美、冷静、一丝不苟。
对校规近乎偏执地维护。老师们都对她赞赏有加，同学们敬畏但亲近不起来。
永远穿着整洁到不自然的制服，头发从不乱一丝。

**真相：** 琴音是汐見学園创始人御崎家的后人。
她的家族代代知道循环的存在，并负责"维护"循环的稳定运行。
她的角色不是打破循环，而是**确保循环不被破坏**。
因为一旦循环崩溃，时间裂隙会扩大，可能吞噬整个地区。

**她的秘密：** 琴音其实**也是残响者**，她记得很多循环。
但她的家族使命要求她压制自己的感情 — 因为每一次感情波动都可能引发循环的不稳定。
她用完美主义的外壳把自己裹得密不透风，夜晚独自在学生会室时会无声地哭泣。
她知道一个残酷的事实：**打破循环的唯一方法，是牺牲一个残响者的存在来封印裂隙。**
而她自己就是最合适的人选。

**核心主题：** 责任与自我的撕裂。当你的存在本身就是为了保护别人而活，
你有权利选择为自己而活吗？完美的表面下，是什么样的空洞？

**线路核心矛盾：**
琴音知道自己必须牺牲自己来封印裂隙（这是她家族的使命）。
但如果主角介入，她面临选择：履行使命独自消失，还是接受主角的帮助寻找其他方法？
其他方法意味着更大的风险 — 可能所有人都无法幸免。

---

### 日向 陽菜（Hinata Hina）— "来自昨日的少女"

**表面：** 转学生。开朗、热情、有点冒失。
说话带着奇怪的口音，对一些现代事物表现出不自然的陌生感。
总是笑得很灿烂，但笑容有时候像是在掩饰什么。
她对主角有一种不合理的亲近感，好像早就认识他一样。

**真相：** 陽菜不是"转入"这个学校的 — 她是**从二十年前穿越过来的**。
她是实验当天恰好在场的一个小女孩（研究员的女儿），
在实验失控时被时间裂隙抛出，随机降落在了某个循环的中间。
对她来说，实验只发生在"昨天"。但客观上已经过了二十年。

**她的秘密：** 陽菜的母亲是研究员之一，实验失控时为了保护女儿把她推进了裂隙。
陽菜一直以为母亲还在"另一边"等她。她表现出的开朗，
是因为她相信只要找到回去的方法就能再见到妈妈。
但随着时间推移，她渐渐意识到：**如果回到实验发生的那一刻，她可能会死在那里。**

**核心主题：** 失去与接受。有些失去无法挽回，那我们要如何带着缺失继续前行？
天真不是无知，是一种选择。

**线路核心矛盾：**
陽菜发现如果能回到实验发生前的时间点，她有机会**阻止实验、拯救所有人（包括她妈妈）**。
但回去意味着她必须再次经历那一天的恐惧，而且成功的概率极低。
更关键的是，阻止实验意味着**循环从未发生** — 她和主角永远不会相遇。

## 剧情结构

### 共通线 · 第一章：「始まりの日」— 开始之日（Day 1）

**场景流：**
1. **清晨** — 主角从宿舍醒来，感觉做了一个很长很长的梦但想不起来内容。窗外是灰蓝色的海。
2. **上学路** — 走上海边坡道，遇到日向陽菜（她热情打招呼，说"总觉得我们好像见过"）。
3. **教室** — 班主任介绍转学生（就是主角）。注意到白瀬雫坐在窗边看窗外。学生会长御崎琴音来教室门口看了一眼就走了。
4. **午休（选项1）** — 去天台/图书馆/学生会室，分别遇到一位女主。
5. **下午** — 放学后听到广播室传来奇怪的静电噪音。调查or忽略（选项2）。
6. **傍晚（选项3）** — 遇到三位女主之一，进行第一次深度对话。
7. **夜晚** — 回宿舍，发现枕头下有一张纸条，字迹是自己的：「这是第██次了。不要忘记她们。」

**本章伏笔/暗示：**
- 主角觉得学校的一切都很熟悉（走廊的转角、教室的门牌号码）
- 雫看到主角时有一瞬间的惊讶，随即恢复平静（她已经见过他无数次了）
- 琴音对主角的档案看得很仔细（她在确认他是否又是"新"的）
- 陽菜对学校的设施表现出"这个我好像见过"的反应
- 广播室的静电噪音其实是旧研究设施的信号残留
- 纸条是上一个循环的自己写的，但他不记得了

---

### 共通线 · 第二章：「綻び」— 裂痕（Day 2-3）

**场景流：**
1. **Day 2 清晨** — 主角醒来时有强烈的既视感（deja vu）。某些对话他似乎能预知对方的下一句话。
2. **调查（选项1）** — 选择与谁一起调查学校的异常。
3. **广播室事件** — 再次听到广播室的噪音。这次如果仔细听，能隐约分辨出是一个女人的声音在念数字（选项2）。
4. **灯塔（选项3）** — 去废弃灯塔调查（与某位女主同行）。在灯塔里发现一张旧照片，上面是几个穿着白大褂的人，其中一人长得像主角的父亲。
5. **旧记录调查（选项4）** — 深入查阅学校旧档案、询问老师、或者独自搜索。
6. **Day 3 夜晚（选项5）** — 三个夜间事件选一个。这是**关键的亲密场景**，每位女主展露脆弱的一面。
7. **Day 3 深夜** — 主角独自在走廊上行走时，**整个世界闪烁了一下** — 像是老旧电视的信号干扰。一瞬间他看到了另一个版本的自己，站在走廊另一端，对他说了一句话后消失。

**本章伏笔/暗示：**
- 广播中的数字是时间坐标
- 旧照片中主角父亲旁边站着一个女人 — 如果是雫线，雫会认出那是她姐姐
- 琴音在夜间场景中哭泣时，如果被撞见会极度慌张并否认
- 陽菜在夜间场景中会说出一些不属于这个时代的话语
- 主角的"既视感"其实是循环记忆在微弱地挣扎

---

### 共通线 · 第三章：「真実」— 真相（Day 4-5）

**场景流：**
1. **Day 4** — 世界开始出现明显的"裂缝"。天空偶尔会变成两个颜色。有学生开始说"今天好像过过了"。
2. **三位女主分别找到主角，各自揭示一部分真相（顺序固定）：**
   - 雫：告诉他循环的存在，展示自己的记录本
   - 琴音：警告他不要深究，透露汀プロジェクト的存在
   - 陽菜：承认自己来自过去，说她"一直在找回去的路"
3. **Day 5 · 抉择（选项1）** — 相信谁的话？（影响后续信息的可信度和路线走向）
4. **Day 5 · 灯塔深处** — 进入灯塔地下的旧研究设施。发现父亲的实验日志。
   日志最后写道：「蓮，如果你在看这段话，说明循环还在继续。
   对不起，我把你留在了这里。但请你记住 — 打破循环的钥匙，
   不是力量，而是选择。三个残响者的羁绊，才是真正的钥匙。」
5. **Day 5 · 对峙（选项2）** — 面对完整真相：要打破循环，需要做出选择。面对or逃避（选项2）。
6. **Day 5 夜晚** — 最后的夜晚。三位女主分别来告别（或者告白）。
7. **Day 6-7** — 根据选择进入对应路线。

**本章伏笔/揭示：**
- 完整真相：打破循环需要至少一个残响者自愿放弃自己的"时间锚点"
- 雫的锚点是她对姐姐的执念
- 琴音的锚点是家族的使命
- 陽菜的锚点是回到过去见母亲的愿望
- 主角自己的锚点是什么？— 是他对"重新开始"的恐惧

### 白瀬雫线 — 「永遠の記録」

**主题：** 记忆与遗忘。记住所有痛苦，还是忘记所有温柔？

**场景流：**
1. **告白** — 雫带主角去图书馆地下室，那里有她数百本循环记录。
   她第一次哭了："我已经……不想再一个人记着了。"
2. **发现方法** — 雫找到了打破循环的方法：通过她的记录重建时间线的"正确坐标"，
   将循环中的人送回到时间裂隙封闭前的状态。代价：所有循环内的记忆消失。
3. **矛盾** — 主角不愿接受这个代价。他们之间的羁绊建立在无数次重复的相遇之上。
   忘记这一切，就等于杀死现在的他们。
4. **分歧选择：**
   - **「循环でもいい」(接受循环)** → Eternal End：两人决定留在循环中。
     雫终于不再独自承担记忆的重量。每个七天，他们重新相遇、重新相爱。
     虽然是永远重复的七天，但只要两个人都记得，这就是属于他们的永远。
     *最后画面：清晨的阳光中，雫合上记录本微笑：「这是第██次了。……但这次，没关系。」*
   - **「終わらせよう」(打破循环)** → True End：主角说服雫，真正的关系不需要靠重复来维系。
     打破循环的瞬间，所有记忆如流沙般消散。
     *最后画面：春天的新学校。雫坐在图书馆窗边，一个男生走过来问"这里有人坐吗？"
     她抬头，眼眶突然红了但不知道为什么。"没有……请坐。"
     他们不记得彼此了。但某种东西还留在那里。*
   - **「第三の道」(第三条路)** → Bittersweet End：主角提出只消除雫一个人的循环记忆负担，
     让她把记录本的内容"写入"世界本身，让所有人都能隐约感知到循环的存在。
     循环没有被打破，但不再是囚笼 — 而是一种温和的提醒。
     *最后画面：学校的钟突然多了一个第八天的刻度。雫看着那个刻度说："这样，至少不会忘记曾经有过七天之外的可能。"*

### 御崎琴音线 — 「完璧な嘘」

**主题：** 完美的谎言与不完美的真实。面具之下，是谁的脸？

**场景流：**
1. **裂缝** — 琴音在主角面前第一次失态。她崩溃着说："你不该来的。
   你不该……让我想起这些。"
2. **使命** — 琴音告诉主角她的家族使命：每隔若干循环，必须有一个残响者
   "融入"裂隙，用自己的存在来修补时间裂缝。她已经被"准备"了整个人生。
3. **矛盾** — 主角不接受她自我牺牲的结局。但琴音反问："如果我不这样做，
   时间裂隙会扩大。你愿意用所有学生的生命来换我一个人的自由吗？"
4. **分歧选择：**
   - **「全員救う」(救所有人)** → True End：主角提出代替琴音融入裂隙 — 他自己也是残响者，
     而且他的存在本身就是循环的"种子"（他父亲的实验创造了他与循环的特殊联系）。
     他融入裂隙后，循环结束。琴音失去了他，但获得了自由。
     *最后画面：琴音第一次没穿制服出现在学校。风吹过她散开的头发。
     她看着天空轻声说："你看，我没有穿制服哦。你看到了吗？"
     但她不知道的是，在某个角落，一个男生站在时间的另一侧，看着她的笑容，微笑着消散。*
   - **「琴音だけを」(只救琴音)** → Bittersweet End：主角强行阻止琴音牺牲，
     用不完整的方法封闭裂隙。循环被打破，但不稳定。
     琴音活了下来，但时间变得"稀薄" — 某些日子会突然消失，
     人们偶尔会丢失几个小时。两人在这不完美的世界中努力活着。
     *最后画面：两人坐在海边的防波堤上。"今天星期几？""不知道。但你在就行了。"*
   - **「見送る」(放手)** → Normal End：主角最终接受了琴音的选择。
     在最后一夜，两人不是以拯救者和被拯救者的身份，而是以两个普通人的身份度过。
     琴音融入裂隙时带着微笑。循环结束后的世界很安静。
     *最后画面：学生会室里，新会长打开抽屉，发现一封没有署名的信：
     「这间屋子的窗台上，曾经有人每天给花浇水。请继续。」*

### 日向陽菜线 — 「昨日への手紙」

**主题：** 回不去的昨天。如果重来一次，你会选择相同的路吗？

**场景流：**
1. **告白** — 陽菜终于对主角说出了全部真相：她来自二十年前。
   "对我来说，那场事故……是昨天发生的事。我的妈妈，还在那边等我。"
2. **发现** — 主角在研究设施中找到了一台旧设备，理论上可以把一个人送回实验发生前的时间点。
   陽菜可以回去，甚至有机会阻止实验。但成功率极低，而且一旦回去就再也回不来。
3. **矛盾** — 如果陽菜成功阻止了实验，循环从未发生，她和主角永远不会相遇。
   但她可以救她的妈妈，可以救所有人。
4. **分歧选择：**
   - **「行かせてあげる」(让她回去)** → Bittersweet End：主角含泪送陽菜回到过去。
     在她消失的瞬间，世界开始改写。循环消散，一切重来。
     *最后画面：一个新的世界线。主角在某个城市的某个街角，
     看到一个穿着旧式校服的女孩在吃冰淇淋。
     她不认识他。但她看到他时，笑着说了句："总觉得我们好像在哪里见过呢。"
     这是被改写的世界线上，命运最后的温柔。*
   - **「受け入れる」(接受失去)** → True End：陽菜决定不回去。
     她说："如果我不回去，妈妈会一直以为我消失了。但至少……她可以慢慢接受。
     如果我回去又失败了，她连希望都不会有了。"
     陽菜选择留在现在，和主角一起打破循环。循环结束后的世界，
     陽菜作为"二十年前失踪的孩子"被找到了。她收到了一封来自母亲的、二十年前写下的信：
     「陽菜，不管你在哪个时代，妈妈都相信你一定能找到幸福。」
     *最后画面：陽菜读完信，靠在主角肩上哭了很久。然后她抬起头，
     笑着说："走吧。今天的天气很好，适合去灯塔看看海。"*
   - **「運命を書き換える」(改写命运)** → Hidden End：主角和陽菜一起回到过去，
     两人合力阻止了实验。但代价是他们被困在了二十年前 — 以小孩的身体。
     *最后画面：二十年前的某个下午。两个小学生坐在海边的防波堤上吃冰棍。
     "我们会重新长大吗？""会的。这一次，从头开始。"
     "那……到时候还要来汐見学園吗？""当然。不过这次，我第一天就要跟你说：你好，我叫蓮。"
     "你好！我叫陽菜！我们做朋友吧！"
     ……他们笑了。远处的灯塔上，钟开始转动了。*

---

### 无路线 · Loop End

条件：第三章结束时未选择任何女主（好感度均不达标 或 玩家主动放弃）。

主角谁都没有选。第七天的午夜到来。一切重置。
下一个循环的第一天，他再次"转学"进汐見学園。
但这一次，他在教室里看到三个女生时，心里空空的 — 连既视感都没有了。

*"他什么都不记得了。她们也什么都不记得了。*
*七天又开始了。和每一次一样。*
*和每一次……一模一样。"*
