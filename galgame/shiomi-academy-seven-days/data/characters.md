# 《汐見学園の七日間》— 角色档案

---

## 主角：長谷川 蓮（Hasegawa Ren）

| 属性 | 内容 |
|------|------|
| 性别 | 男 |
| 年级 | 高中二年级（外表） |
| 真实年龄 | 不明（已在循环中度过远超认知的时间） |
| 身份 | 长谷川诚一之子，残响者 |

**性格特点：**
- 表面是普通、温和的转学生，有点沉默
- 内心偶尔会有强烈的既视感，但总是归结为"想多了"
- 对三位女主有本能的保护欲（来自深层记忆的残留）
- 不擅长表达感情，但行动力强

**对话风格：**
- 第一人称，内心独白多
- 语言简洁，不废话
- 偶尔会说出不该知道的事情（循环记忆碎片）
- 对雫：温柔但小心翼翼
- 对琴音：敬佩但想穿透她的面具
- 对陽菜：觉得她像太阳，自己容易被吸引

**背景：**
- 二十年前实验失控时是5岁的孩子
- 父亲将他送入循环保护他
- 每个循环的第一天都会"转学"进学校
- 不知道自己的真实身份

---

## 女主一：白瀬 雫（Shirase Shizuku）— "记忆的守望者"

| 属性 | 内容 |
|------|------|
| 性别 | 女 |
| 年级 | 高中二年级 |
| 身份 | 图书委员，残响者（记忆最完整） |
| 秘密 | 姐姐白瀬響被困在时间裂隙另一侧 |

**外貌：**
- 黑色长发，通常披着或扎低马尾
- 眼睛是深棕色，偶尔在光线下显得像琥珀
- 总是穿着整洁但略显陈旧的校服
- 手指纤细，指尖有墨渍

**性格特点：**
- 安静寡言，说话声音轻柔
- 眼神偶尔流露出与年龄不符的疲惫和哀伤
- 对书籍有深厚的感情（它们是她记忆的锚点）
- 害怕主动告知真相（因为别人会忘记）
- 内心深处渴望有人陪她一起记住

**对话风格：**
- 说话简洁，用词精确
- 很少用感叹号，语气平淡
- 经常引用书中的话（但不说是哪本书）
- 偶尔会说出"你又来了"这样的话，然后立刻改口
- 对主角：克制但深情，像是怕靠太近又会失去

**经典台词：**
- "……你已经问过这个问题了。"（第一次见面时不小心说漏）
- "记忆是诅咒还是祝福，取决于你是否有人分享。"
- "这本书……我读过很多次了。但每次读，感觉都不一样。"
- "我不想再一个人记着了。"

**与主角的关系发展：**
- 初期：警惕但忍不住关心（她知道他每次都会忘记）
- 中期：逐渐信任，分享更多真相
- 后期：告白（在图书馆地下室，展示数百本记录本）
- 线路核心：记忆与遗忘的选择

---

## 女主二：御崎 琴音（Misaki Kotone）— "秩序的守护者"

| 属性 | 内容 |
|------|------|
| 性别 | 女 |
| 年级 | 高中三年级 |
| 身份 | 学生会长，御崎家族后人，残响者 |
| 秘密 | 家族使命要求她牺牲自己封印裂隙 |

**外貌：**
- 棕色短发，整齐到不自然
- 眼睛是深灰色，锐利但偶尔流露脆弱
- 永远穿着完美的制服，头发从不乱
- 站姿笔直，像是时刻在维持形象

**性格特点：**
- 表面完美、冷静、一丝不苟
- 对校规近乎偏执地维护（实际上是在维持循环稳定）
- 用完美主义的外壳把自己裹得密不透风
- 夜晚独自在学生会室时会无声地哭泣
- 内心深处渴望有人看穿她的面具

**对话风格：**
- 语气正式，用敬语（即使对同级生）
- 很少表达个人意见，总是"作为学生会长"
- 偶尔会说出"你不该问这个"（因为她知道真相会带来痛苦）
- 被戳中软肋时会慌张、否认、甚至愤怒
- 对主角：抗拒但逐渐被他的真诚打动

**经典台词：**
- "校规是为了保护所有人而存在的。请遵守。"
- "你不该来的。你不该……让我想起这些。"
- "如果我不这样做，时间裂隙会扩大。你愿意用所有学生的生命来换我一个人的自由吗？"
- "……你看，我没有穿制服哦。你看到了吗？"

**与主角的关系发展：**
- 初期：警惕、抗拒（她不想让他介入）
- 中期：被他的坚持打动，逐渐展露脆弱
- 后期：告白（在夜晚的学生会室，第一次失态哭泣）
- 线路核心：责任与自我的撕裂

## 女主三：日向 陽菜（Hinata Hina）— "来自昨日的少女"

| 属性 | 内容 |
|------|------|
| 性别 | 女 |
| 年级 | 高中二年级（外表） |
| 真实年龄 | 20年前的小女孩（约7岁），被困在循环中 |
| 身份 | 研究员之女，从20年前穿越而来 |
| 秘密 | 母亲为了保护她把她推进了时间裂隙 |

**外貌：**
- 茶色短发，有点乱，显得活泼
- 眼睛是明亮的棕色，总是带着笑意
- 校服穿得不太整齐，领结经常歪
- 笑起来很灿烂，但有时候笑容像是在掩饰什么

**性格特点：**
- 开朗、热情、有点冒失
- 说话带着奇怪的口音（20年前的方言）
- 对一些现代事物表现出不自然的陌生感（智能手机、网络用语）
- 总是笑得很灿烂，但笑容有时候像是在掩饰恐惧
- 对主角有一种不合理的亲近感（她"昨天"见过他）

**对话风格：**
- 语气活泼，感叹号多
- 经常说"总觉得我们好像见过"
- 会用一些过时的词汇或表达方式
- 偶尔会说出"妈妈说过……"然后突然停住
- 对主角：天然亲近，像是早就认识他

**经典台词：**
- "总觉得我们好像在哪里见过呢！"
- "这个……我好像见过。但应该不可能吧？"
- "对我来说，那场事故……是昨天发生的事。我的妈妈，还在那边等我。"
- "你好！我叫陽菜！我们做朋友吧！"

**与主角的关系发展：**
- 初期：天然亲近（她记得"昨天"的他）
- 中期：逐渐信任，分享自己的秘密
- 后期：告白（在灯塔下，说出全部真相）
- 线路核心：失去与接受的选择

## 重要配角

### 长谷川 诚一（Hasegawa Seiichi）— 主角的父亲

| 属性 | 内容 |
|------|------|
| 身份 | 物理学家，汀プロジェクト主持人 |
| 状态 | 实验失控后失踪（实际被困在时间裂隙中） |

**背景：**
- 二十年前主持时间折叠实验
- 实验失控时将儿子蓮送入循环保护他
- 在灯塔地下留下了实验日志
- 日志最后写道：「蓮，如果你在看这段话，说明循环还在继续。
  对不起，我把你留在了这里。但请你记住 — 打破循环的钥匙，
  不是力量，而是选择。三个残响者的羁绊，才是真正的钥匙。」

**在游戏中的出现：**
- 只在灯塔地下的实验日志中以文字形式出现
- 旧照片中有他的影像（主角认出他像自己的父亲）

### 白瀬 響（Shirase Hibiki）— 雫的姐姐

| 属性 | 内容 |
|------|------|
| 身份 | 实验助理研究员 |
| 状态 | 实验失控时消失，被困在时间裂隙"另一侧" |

**在游戏中的出现：**
- 只在旧照片和雫的回忆中出现
- 是雫的核心动机之一

### 日向 美咲（Hinata Misaki）— 陽菜的母亲

| 属性 | 内容 |
|------|------|
| 身份 | 研究员之一 |
| 状态 | 实验失控时为了保护女儿把她推进了裂隙 |

- 陽菜一直以为母亲还在"另一边"等她

## 残响者（Zankyou）设定

**定义：** 极少数能在循环中保留记忆碎片的人。

**已知残响者：**
1. **白瀬雫** — 记忆保留最完整，记得每一个循环
2. **御崎琴音** — 记得很多循环，但被家族使命压制感情
3. **日向陽菜** — 从20年前穿越而来，记忆完整但不属于这个循环
4. **長谷川蓮** — 记忆几乎完全流失，但偶尔会有既视感

**残响者的共同特征：**
- 能在循环中保留部分或全部记忆
- 感情波动可能引发循环不稳定
- 是打破循环的关键（需要自愿放弃"时间锚点"）

---

## 时间锚点（Temporal Anchor）设定

**定义：** 每个残响者内心最深处的执念，是他们与循环的联系。

**三位女主的锚点：**
1. **雫** — 对姐姐白瀬響的执念（想救她）
2. **琴音** — 家族的使命（必须牺牲自己封印裂隙）
3. **陽菜** — 回到过去见母亲的愿望

**主角的锚点：**
- 对"重新开始"的恐惧（他害怕打破循环后再次失去一切）

**打破循环的条件：**
- 至少一个残响者自愿放弃自己的时间锚点
- 三个残响者的羁绊是"钥匙"（需要情感共鸣）
