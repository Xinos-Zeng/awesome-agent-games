# 敌人表 Enemies

## 战斗规则说明
- 所有战斗调用 `scripts/combat.py`
- 敌人行动由脚本随机决定，Agent 负责叙述过程
- HP归零 = 战斗结束（敌人倒下或逃跑）
- 玩家HP归零 = 游戏结束（结局：死亡）

---

## 普通敌人

### cultist — 邪教徒残余
- HP：8
- 攻击力：1d4（拳打脚踢）或 1d6（持刀）
- 防御：无（AC=0，普通命中检定即可）
- 理智扣减：遭遇时无（普通人类）；看到其进行仪式时 1d3
- 行动模式：50%普通攻击，30%防御（本回合伤害-2），20%逃跑
- 特殊：被击败后可搜身，有概率找到【old_key】（如果尚未获得）
- 叙事定位：前科比特教徒，仍在守护这栋建筑

---

## 神话存在（不可直接战胜）

### corbitt_soul — 科比特之魂
- HP：∞（普通武器无效，仪式匕首可伤害，HP=20）
- 攻击力：
  - 心灵撕裂：不造成HP伤害，直接扣 1d6 理智
  - 幽灵之手：造成 1d4 HP伤害（只有理智<30时才使用）
- 防御：免疫普通武器；对仪式匕首正常受伤
- 理智扣减：
  - 初次目击：理智检定，失败扣 1d8，成功扣 1
  - 每回合持续凝视：额外扣 1 理智
- 行动模式：70%心灵撕裂，30%幽灵之手
- 特殊规则：
  - 若玩家持有【strange_manuscript】并选择"执行驱逐仪式"，战斗立即结束（结局A）
  - 若玩家理智归零，战斗结束（结局B）
  - 若玩家选择逃跑，科比特之魂不追击，但会在玩家逃跑前再施放一次心灵撕裂
- 叙事定位：沃尔特·科比特，已死的邪教徒，灵魂被封印在公寓地下室，通过折磨租客来维持存在

---

## 战斗流程（Agent执行顺序）

```
1. 宣布战斗开始，显示双方状态
2. 玩家选择行动
3. 调用 scripts/combat.py 结算本回合
4. Agent 用叙事语言描述结果（不要直接念数字）
5. 更新并显示当前状态
6. 判断战斗是否结束
```

## 战斗状态显示模板

```
┌────────────────────────────────────┐
│  你                    [敌人名]    │
│  HP: [当前]/[最大]     HP: ████    │
│  理智: [当前]          状态: 正常  │
└────────────────────────────────────┘

可用行动：
▶ [攻击]   掷d20命中检定，成功则造成武器伤害
▶ [道具]   使用背包中的道具
▶ [逃跑]   【敏捷】检定，成功脱战，失败损失1d4 HP
```
