# 主线剧情总纲 Story

## 故事背景

**时间：** 1923年10月，深秋
**地点：** 美国马萨诸塞州，波士顿，克劳福德街14号，布伦纳公寓

**玩家身份：** 私家侦探（或记者、学者，由玩家创建角色时决定）

**委托：** 公寓房东雇佣玩家调查布伦纳公寓。过去一年，连续三名租客在此
精神崩溃，无人敢再租住。房东付50美元，要求查明原因。

---

## 真相（KP/Agent掌握，玩家需调查发现）

沃尔特·科比特，1913年前的租户，是秘密团体"繁星之门"的成员。
该团体从事召唤仪式，试图与神话存在沟通。

1913年，科比特独自在公寓地下室完成了一个残缺的封印仪式——
他的意图是将自己的灵魂封印于此，以获得"永生"。
仪式成功了，但不完整：他的身体死去，灵魂被困在地下室，
无法离开，无法消散，痛苦而扭曲。

十年来，每一个住进公寓的人都受到他灵魂的侵扰，在梦中被折磨，
最终精神崩溃。科比特并非出于恶意——他在寻求解脱，
但他唯一会的只有恐惧和折磨。

**解脱方式：** 用密室中的手稿执行"驱逐仪式"——
这实际上是完整版的封印解除仪式，能让科比特的灵魂最终消散。

## ⏰ 黑夜倒计时系统

游戏开始时隐藏时钟启动，**每次主要行动消耗1点时间**，满10点后黑夜降临。

**计入时间消耗的行动：**
- 前往新场景（每次+1）
- 与NPC深度交谈（+1）
- 在同一场景内进行第2次及以后的搜查（+1）
- 检定失败后重试（+1）

**不计入时间的行动：**
- 第一次进入场景
- 拿取已明显可见的道具
- 选择离开场景

**黑夜降临后的效果：**
- 地下室：邪教徒必然出现（无论是否获得钥匙），且数量变为2人
- 楼上：科比特的干扰开始渗透到地面楼层，每次在公寓内行动需额外理智检定（难度25），失败扣1d2
- 部分NPC：陈太太和莱利警官不再开门/接待
- 氛围叙述：Agent应强化黑夜氛围，路灯昏黄、街道清冷

**时钟由 game_state.json → time_elapsed 字段维护，Agent每次行动后调用 transition.py tick 更新。**

## 线索互斥规则

**警察局 OR 图书馆 只能深度探索一个：**
- 时间有限，第一个深度探索的地方正常进行
- 第二个地方只能获取基础信息（NPC说"时间晚了，改天再来"）
- Agent需追踪 flags: deep_investigated_police / deep_investigated_library

**陈太太说服失败的代价：**
- 她关门，不再开门
- old_key 只能通过击败邪教徒搜身获得
- 地下室铁门只能用开锁检定（难度65，更难）

**书桌/衣柜搜查各只有一次机会：**
- 检定失败 → 该搜查点永久关闭，不能重试

## 最终对决强化：三段仪式

拿到手稿后，执行驱逐仪式**需要念完三段咒文**，不能一次完成：

```
第一段咒文：科比特之魂出现，发动一次心灵撕裂
  → 理智检定（难度35），失败扣1d6，成功扣1d2
  → 玩家选择：继续念/停止逃跑

第二段咒文：科比特之魂形态开始动摇，发动一次幽灵之手
  → HP伤害 1d4，理智检定（难度35），失败扣1d6，成功扣1d2
  → 玩家选择：继续念/停止逃跑

第三段咒文：最后一段，科比特之魂垂死挣扎，发动最强攻击
  → 理智检定（难度25），失败扣1d8，成功扣1d3
  → 念完即触发结局A
```

中途任意阶段选择停止 → 结局C
中途理智归零 → 结局B
中途HP归零 → 新结局D（死亡）

## 设计备注

- old_key 三种获得路径：陈太太给 / 击败邪教徒搜身 / 开锁检定（难度65）
- 结局A需要：strange_manuscript（必须）+ 撑过三段仪式
- 结局A（战斗路线）需要：ritual_dagger（必须）+ 足够HP和理智
- 结局C/D随时可触发
- 黑夜倒计时是软压力，不会强制结束游戏，但会让后半段更危险
- 警察局/图书馆互斥迫使玩家做路线选择，增加重玩价值
